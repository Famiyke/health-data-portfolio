How Every Project Is Built
Step 01
Source Verification
Every dataset is sourced from a named Canadian government or agency publication. Download URLs are documented in each project README. No data is invented without anchoring to published figures.
Step 02
Data Cleaning and Profiling
Before any analysis, a dedicated cleaning script checks for missing values, out-of-range indicators, suppressed cells, and coverage gaps. All decisions are logged with rationale.
Step 03
SQL and Python Analysis
Core questions are answered in SQL for reproducibility and extended in Python for depth. Every finding is verifiable from the raw data. No black-box models — only readable, explainable logic.
Step 04
Visualisation and Communication
Dashboards are built for a non-technical audience. Every chart has a single clear message. Projects include a written brief a health director can read in 10 minutes and act on.
